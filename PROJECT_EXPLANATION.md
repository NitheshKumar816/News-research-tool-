# News Research Tool - Complete Project Explanation

## 🎯 Project Overview
**Streamlit RAG Application** for analyzing news articles with **stock research focus**. 

**Purpose**: Load news URLs → Extract stock targets (₹100-5000), recommendations (Buy/Sell), Q&A, summarization.

**Tech Stack**:
```
Frontend: Streamlit
Embeddings: sentence-transformers/all-MiniLM-L6-v2
Vector DB: FAISS
LLM: google/flan-t5-large (local)
Loader: newspaper3k
```

## 🚀 Step-by-Step Code Flow

### 1. **UI Setup** (`st.set_page_config`, sidebar)
```
URLs input (max 3) → \"Process URLs\" button
Main placeholder for progress
```

### 2. **Model Loading** (`@st.cache_resource`)
```
embeddings = HuggingFaceEmbeddings(\"sentence-transformers/all-MiniLM-L6-v2\")
llm = HuggingFacePipeline(\"google/flan-t5-large\", max_new_tokens=256)
```

### 3. **URL Processing** (on button click)
```
NewsURLLoader(urls) → raw documents
Clean: min 500 chars → add source metadata
RecursiveCharacterTextSplitter(chunk_size=800, overlap=200)
FAISS.from_documents(docs, embeddings) → save_local(\"faiss_index\")
```

### 4. **Query Processing** (text_input + search)
```
Load FAISS → retriever (k=20 similarity)
Route based on query keywords:
  ├─ \"target/price/stock\" → Regex: Rs./₹[100-5000]
  ├─ \"recommend/buy/sell\" → Regex: Buy|Accumulate|Sell  
  └─ Else → RetrievalQAWithSourcesChain(llm + retriever)
```

### 5. **Stock Target Extraction** (Precise Regex)
```python
numbers = re.findall(r'(?:Rs\.?|₹)\s*([\d,]{3,6})', full_text)
→ Remove commas → Filter 100-5000 → Sorted unique → \"1500, 2500\"
```

### 6. **Display + Debug**
```
Answer → Expander with raw retrieved chunks
Chat history in session_state
\"Summarize Articles\" button → Top-5 chunks → LLM summary
```
